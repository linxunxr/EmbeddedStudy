# 命令
## 1 命令的格式
### 1.1 打开终端的方式
使用快捷键：`Ctrl + alt + t`(Ubuntu)
### 1.2 终端提示符
`linxun@ubuntu:~$`
- linxun:用户名
- ubuntu：主机名
- ~：表示路径
  - ~：表示家目录
  - /：表示根目录
- $:表示权限
  - $：普通用户权限
  - #：管理员权限
### 1.3 命令的格式
- 命令 选项 参数
- 注意：命令、选项、参数之间必须用空格隔开
- 例如：`ls -l`
## 2 命令帮助信息
- `命令 --help`
- 注意：并不是所有的命令都有帮助信息
## 3 man命令
- man命令是Linux提供的帮助手册，可以查询命令、函数或者特殊的文件
- 第一章中放的是 命令 的帮助信息
- 第二章中放的是 系统调用：操作系统提供的函数open
- 第三章中放的是 库函数：c语言库提供的函数printf 的帮助信息
- 用法：`man 章节 查找信息`
## 4 命令常用的技巧
### 4.1 自动补全
- 输入命令的一部分，然后按`tab`键，就会自动补全命令，如果有多个相同开头的命令，多按一次tab键，会将所有的以已经输入开头的命令提示出来
### 4.2 历史命令
- 可以按 上下键 将以前输入的命令调出来，回车执行或者修改之后回车执行
### 4.3 重定向
- `命令 > 目标文件` 先清空原本文件内容
- `命令 >> 目标文件` 以追加的形式写入
- 将执行命令输出到屏幕上的数据，重定向输出到 目标文件 中
### 4.4 管道
- 一个程序的输出 作为另外一个程序的输入
- `ls --help | more` 以每页的方式查看ls命令的帮助信息
- `ls /etc | wc -w` 同意/etc目录下的文件数
## 5 ls tree clear
### 5.1 ls
- `ls` 查看当前目录下的文件名
- `ls  目录名` 查看指定目录下的文件名
- `ls -a` 查看当前目录下文件的详细信息
- `ls -l` 查看当前目录下文件的详细信息
  - linux里面一共有七种文件类型`bcd-lsp`
  - b：块设备文件
  - c：字符设备文件
  - d：目录文件
  - -：普通文件
  - l：软连接文件
  - s：套接字文件
  - p：管道文件
- `ls -al` 查看当前为目录下所有文件的详细信息
- `ls -hl` 详细信息中的字节数可以带单位地显示
### 5.2 tree
- 以树状结构显示目录信息
- 系统默认未安装 需要安装 `sudo apt-get install tree`
- `tree -L 3` 显示三层目录
### 5.3 clear
- 清屏操作
- `ctrl+l` 也可以执行清屏操作
## 6 cd pwd
### 6.1 cd
- 改变目录
- `cd /home/...` 以绝对路径方式改变目录
- `cd ./...` 以相对路径方式改变目录
- `cd -` 回到刚才所在的路径
- `cd ..` 返回上一级路径
### 6.2 pwd
- 显示当前工作目录的绝对路径
## 7 cat rm cp mv
### 7.1 cat
- 显示文本文件内容
- `cat 文件路径+文件名称/文件名称`
### 7.2 rm
- 删除文件
- `rm 选项 文件夹或文件路径+文件名称或文件名称`
- 如果删除文件夹选项比如为`-r`或`-rf`
- `-f` 为强制删除
### 7.3 cp
- 复制
- `cp 选项 原文件或文件路径+文件名 目的路径`
- 拷贝文件夹时必须加`-r`
- `-v` 显示拷贝进度
### 7.4 mv
- 移动
- `mv 选项 要移动的文件或目录 目的路径`
- 如果原文件和目的路径在同一路径下，相当于重命名，如`mv test.txt test01.txt`
## 8 mkdir touch
### 8.1 mkdir
- 创建文件夹
- `mkdir 文件夹名称`
### 8.2 touch
- 创建文件
- `touch 文件名`
## 9 find
- 在执行路径下查找指定文件
- `find 路径 -name 文件名`
## 10 grep
- 在指定文件中查找指定内容
- `grep 查找信息 文件名 参数-n` 显示行号
## 11 ln
- 创建链接文件
- `ln 原文件名称 链接文件名称 -s`
## 12 tar
- 压缩与解压缩命令
### 12.1 gzip格式
- 压缩：`tar zcvf 压缩包包名 文件1 文件2 ...`
- 解压方法1：`tar zxvf 压缩包包名`
- 解压方式2：`tar zxvf 压缩包包名 -C 目标路径`
### 12.2 bz2格式
- 压缩：`tar jcvf 压缩包包名 文件1 文件2 ...`
- 解压方式1：`tar jxvf 压缩包包名`
- 解压方式2：`tar jxvf 压缩包包名 -C 目标路径`
## 13 vi编辑器
### 13.1 安装vim
- 安装 vim：`sudo apt-get install vim`
- 安装ctags：`sudo apt-get install ctags`
- 配置vim
### 13.2 使用vi
- 使用vi打开/或新建文件，并将光标置于第一行行首：`vi 文件名`
- 使用vi打开文件，并将光标置于第n行行首：`vi +n 文件名`
### 13.3 vi的三种模式
- 插入模式：可以直接编辑文档
- 编辑模式：在编辑模式可以输入一些命令，例如：复制n行、剪切n行等
- 命令模式：可以保存文件、退出vi等
  - `:w`：保存文件
  - `:wq`：保存文件并退出vi
  - `:x`：保存文件并退出vi
  - `:q!`：不保存文件，强制退出vi
  - `:w 文件名`：另存为文件名
- 三种模式之间的切换：
  - 由编辑模式->插入模式
    - `a`
    - `i`
    - `o`
  - 由插入模式->编辑模式：ESC
  - 由编辑模式->命令模式：shift
## 14 编译器gcc
- 编译分为四个阶段：从.c->应用程序
  - 预编译
  - 编译
  - 汇编
  - 链接
- 一步到位：`gcc hello.c -o hello`，运行程序：`./hello`
- 分步骤
  - 预编译：`gcc -E hello.c -o hello.i`
  - 编译：`gcc -S hello.i -o hello.s`
  - 汇编：`gcc -c hello.s -o hello.o`
  - 链接：`gcc hello.o -o hello`
# Makefile
## 1 make
- make是个命令，用来解析Makefile文件的命令
## 2 Makefile语法及其执行
### 2.1 Makefile语法规则
- 目标：依赖文件列表
- \<Tab\>命令列表
- Makefile示例：
```
main:main.c main.h
  gcc main.c -o main
clean:
  rm main
```
### 2.2 make命令格式
- `make [-f file] [targets]`
- `[-f file]`：make默认在工作目录中寻找 GUNmakefile makefile Makefile 文件作为makefile的输入文件，`-f`可以指定除了以上名字之外的文件作为makefile输入文件
- `[targets]`：指定了make工具要实现的目标
